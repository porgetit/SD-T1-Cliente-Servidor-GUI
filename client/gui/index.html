Ayuda de comandos

Comandos Disponibles

Comando Descripción
list Lista todos los usuarios conectados al servidor.
chat:user Inicia o cambia a una sesión de chat con 'user'.
sessions Muestra tus chats activos actualmente.
stop Finaliza el chat con el usuario actual.
stop:user Finaliza específicamente el chat con 'user'.
file Abre un diálogo para seleccionar archivos y enviarlos al chat actual.
accept Acepta una solicitud de chat entrante.
deny Rechaza una solicitud de chat entrante.
exit Cierra la aplicación del cliente.
